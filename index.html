📖 关于
WORD-FORCE
强制单词 · 词汇强制记忆工具

一款移动端优先的单词学习应用，通过选择题、拼写和填空拼写三种题型强制巩固词汇记忆。

特性：

  • • 8 级熟练度系统（Lv.0–Lv.7）
  • • 6 种学习模式：普通 / 复习 / 学习 / 错题深渊 / 闪电战 / ☠️ 炼狱
  • • ⚔️ 单词 PK 模式：横屏双人分屏对战
  • • 自定义词库 JSON 导入
  • • 虚拟键盘，专注拼写
  • • 耻辱榜记录高频错词
  • • 每关前 60s 预习本关单词
  • • 排行榜 / 关于页面
