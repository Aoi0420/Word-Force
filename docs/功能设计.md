# Word-Force (强制单词) — 功能设计文档

> 版本：v1.0  
> 核心理念：没有花哨的记忆方法，只有强制的重复、死记硬背

---

## 1. 产品定位

一款**极简、残酷**的记单词工具。不提供词根词缀、联想记忆、趣味图片等花哨功能，只做一件事：**轮番轰炸你的眼睛和手，直到单词刻进脑子里。**

目标用户：需要大量、快速积累词汇的应试考生（考研、托福、GRE），以及信奉"重复是记忆之母"的学习者。

---

## 2. 核心功能模块

### 2.1 词库管理

| 功能 | 说明 |
|------|------|
| 导入 JSON | 用户上传符合 `{"en":"abandon","pos":"v","zh":"遗弃、放弃"}` 格式的 JSON 文件，作为当前词库 |
| 内置示例词库 | 首次使用自动加载内置的 50 个高考/四级高频词作为示例 |
| 词库信息面板 | 显示当前词库的单词总数、已掌握数、未学数 |
| 词库切换 | 支持多词库本地存储及切换（未来可通过 Cookie `wordlist_index` 扩展） |

### 2.2 学习模式

核心机制：**"三档制"记忆状态**（这是整个应用的核心）

每个单词有且仅有三种状态：

| 状态 | 含义 | 触发条件 |
|------|------|----------|
| **未学（new）** | 还没学过 | 第一次载入词库 |
| **学习中（learning）** | 正在强制重复 | 用户至少答对 1 次，但尚未达到掌握阈值 |
| **已掌握（mastered）** | 暂时不用再复习 | 连续答对 N 次（N 默认为 3，可配置） |

> **关键规则**：任何一次答错，该单词立即**降级回"学习中"状态**，已累计的连续答对次数归零。不设"模糊"或"半掌握"状态，残酷但清晰。

#### 模式一：英文→中文（看英文选中文）

- 展示一个英文单词（含词性）
- 显示 4 个中文释义选项（1 个正确 + 3 个干扰项）
- 干扰项从当前词库中随机抽取
- 点击选项后立即显示对错
- **答对**：该单词连续答对数 +1
- **答错**：连续答对归零，单词降级

#### 模式二：中文→英文（看中文拼英文）

- 展示中文释义 + 词性
- 输入框要求用户输入对应的英文单词
- 提交后对比答案（不区分大小写）
- **答对**：连续答对数 +1
- **答错**：连续答对归零

#### 模式三：听写模式（未来扩展）

- 使用浏览器语音合成（TTS）朗读英文
- 用户输入听到的单词

### 2.3 强制复习机制

这是"强制"二字的灵魂所在：

- **闯关制**：每次学习为一个"关卡"，包含 10 个单词（可配置）
- 关卡未完成（10 个单词全部达到当日目标）前，**不能退出**
- 复习频率算法（非间隔重复，而是强制轮询）：
  - 每学习 5 个新词，强制插入 2 个之前答错的旧词
  - 已掌握的单词每过 24 小时降级为"学习中"，需要重新巩固
  - 用户不能跳过或标记"太简单"
- **惩罚机制**（可选）：连续答错 3 次同个单词，该词会以更短间隔再次出现

### 2.4 学习统计

| 指标 | 说明 |
|------|------|
| 今日学习量 | 今天一共学过多少个不同单词 |
| 今日掌握量 | 今日新掌握的单词数 |
| 当前连击 | 本次学习连续答对次数 |
| 总掌握进度 | 已掌握 / 词库总数 |
| 最常出错 | 答错次数最多的 5 个单词（耻辱榜） |
| 学习时长 | 本次学习用时 |

### 2.5 设置项

| 设置 | 默认值 | 说明 |
|------|--------|------|
| 掌握阈值 | 3 | 连续答对几次算"已掌握" |
| 每关单词数 | 10 | 每轮学习的单词数量 |
| 每日新词上限 | 20 | 每天最多学多少个新词 |
| 复习强度 | 中等 | 低/中/高，影响旧词混入频率 |
| 干扰项数量 | 3 | 选择题的干扰项个数 |
| 惩罚倍数 | 1x | 答错后该词出现的权重倍数 |

## 3. 用户流程

```
启动应用
  │
  ├── [首次使用] 自动加载内置示例词库
  │
  ├── 导入/选择词库 ──→ 查看词库信息
  │
  ├── 开始学习
  │    │
  │    ├── 选择学习模式 (英→中 / 中→英)
  │    │
  │    ├── 进入关卡循环
  │    │    │
  │    │    ├── 取出下一个单词
  │    │    ├── 展示题目
  │    │    ├── 用户作答
  │    │    ├── 反馈对错 + 显示正确答案
  │    │    ├── 更新单词状态
  │    │    └── 是否完成关卡？──→ 否 → 继续
  │    │         │
  │    │         └── 是 → 关卡结算
  │    │               ├── 正确率
  │    │               ├── 连击记录
  │    │               └── 鼓励/嘲讽文案
  │    │
  │    └── [退出学习] 返回首页
  │
  ├── 查看统计
  │
  └── 设置
```

## 5. 非功能需求

- 所有数据存储在 Cookie 中，用户无需注册
- 单页面应用，离线可运行
- 响应式设计，支持移动端使用
- 界面风格：暗色主题、冷酷、军事化 UI 风格（符合"强制"调性）
- 每次刷新后自动恢复到上次学习位置
